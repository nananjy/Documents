# Apollo介绍
Apollo（阿波罗）是携程框架部门研发的开源配置管理中心，能够集中化管理应用不同环境、不同集群的配置，配置修改后能够实时推送到应用端，并且具备规范的权限、流程治理等特性。

Apollo支持4个维度管理Key-Value格式的配置：
 - application (应用)
 - environment (环境)
 - cluster (集群)
 - namespace (命名空间)
